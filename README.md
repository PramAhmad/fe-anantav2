# Ananta (Remake React/Next.js)

Project ini adalah hasil remake dari versi lama:
https://github.com/PramAhmad/ananta

Versi saya migrasi ke React (Next.js) supaya struktur frontend lebih rapi, lebih gampang dikembangin, dan mudah untuk di scale, seeta fokus ke dalam front end di web tidak pwa.

## Inti Project

Ananta fokus bantu komunikasi yang lebih inklusif antara dua arah tuna rungu ke non-tuna rungu dan sebaliknya.

## Fitur yang ada sekarang

- Beranda: halaman utama untuk masuk ke semua fitur.
- Translator Gestur: halaman untuk alur terjemahan gestur ke bahasa isyarat.
- Translator Suara: halaman untuk  terjemahan suara ke bahasa isyarat.
- Edukasi Isyarat: kumpulan materi belajar bahasa isyarat berbentuk video.



## Jalankan di lokal

```bash
npm install
npm run dev
```

Lalu buka: http://localhost:3000
